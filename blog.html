Important FAQ

What is the Document Object Model (DOM) ?
The Document Object Model (DOM) is a programming interface that represents a web page's structure as a tree of objects.
How do you select and element from the DOM?
Use document.querySelector() or document.getElementById().
What is event delegation in the context of the DOM, and why is it useful?
Event delegation is using a parent element to handle events for its child elements. It's useful for better performance and handling dynamic content.
How do you manipulate an element's attribute and styles using the DOM?
Use .setAttribute() for attributes and .style for styles.
